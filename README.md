# parcel_tutorial 🚀📦
Boilerplate for making modules and mocks

***

## Setup 🛠

```
$ git clone https://github.com/t--takai/parcel_tutorial.git
$ cd parcel_tutorial
$ npm install
```

## Running 🚀

```
$ npm test
```
